pg
просмотр файлов на экране


СИНТАКСИС
  
pg [опции] файл1 файл2 файл3

ОПИСАНИЕ
  Команда pg позволяет просматривать файлы страницами, равными по размеру экрану. Если имена файлов не заданы или задан символ -, то просматривается стандартный ввод. После вывода очередной страницы следует приглашение. Если пользователь нажмет клавишу Enter, то будет выведена очередная страница; остальные возможности перечислены ниже. Эта команда позволяет возвращаться по тексту назад (см.ниже).

ОПЦИИ
 
-число число задает размер (в строках) страницы (по умолчанию - 23)
-p приглашение изменение текста приглашения; строка приглашения содержит %d, то при выводе первая пара символов %d будет заменена номером текущей страницы
-f подавление расщепления строк
+номер_строки указание номера_строки, с которого следует начинать просмотр
+/шаблон/ указание шаблона для поиска строки, с которой начинается просмотр, шаблон задается регулярным выражением

Команды
  Когда команда pg делает паузу между страницами, пользователь может вводить команды одного из трех типов: просмотра, поиска и изменения режима.

Перед командами просмотра может задаваться адрес - число, возможно со знаком, показывающее, с какого места файла продолжать просмотр. Этот адрес, в зависимости от команды, интерпретируется в единицах либо страниц, либо строк. Адрес со знаком задает точку относительно текущей страницы или строки, а адрес без знака - относительно начала файла.

Команды просмотра
(+1)перевод_строки вывод на экран очередной страницы; адрес задается в страницах
(+1)l движение по файлу на заданное количество строк
(+1)d
CTRL+D		 перемещение на полэкрана вперед или назад
CTRL+L повторный вывод текущей страницы текста.
$ вывод последней полной страницы файла

Команды поиска
i/шаблон/ поиск вперед до i-го (по умолчанию i=1<.samp>) появления шаблона; поиск начинается сразу после текущей страницы и продолжается до конца файла
i^шаблон^
i?шаблон?		 поиск назад до i-го (по умолчанию i=1) появления шаблона; писк начинается сразу перед текущей страницей и продолжается до начала файла

Команды модификации режима
in перейти к i-му следующему файлу командной строки (по умолчанию i=1
ip перейти к i-му предыдущему файлу командной строки (по умолчанию i=1)
iw продвижение по файлу на размер окна
s файл сохранение входной информацию в заданном файле
h вывод справки о доступных командах
q завершение просмотра файлов
!команда команда передается интерпретатору, имя которого представляет собой значение переменной окружения SHELL

В процессе вывода на терминал пользователь может в любой момент нажать клавиши завершения или прерывания; в результате этого вывод будет прекращен.
