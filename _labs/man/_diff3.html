diff3
поиск различий между тремя файлами


СИНТАКСИС
  
diff3 [опции] файл1 файл2 файл3

ОПИСАНИЕ
  Команда diff3 сравнивает содержимое трех файлов и выводит описание различий между ними. Одно из имен файлов может быть - (знак "минус"), это означает, что текст должен читаться из стандартного ввода.

Команда diff3 выдает различающиеся фрагменты текста, помеченные следующим образом:
==== Все три файла различны.
====1 Отличия в файле1.
====2 Отличия в файле2.
====3 Отличия в файле3.

Изменения, которые необходимо произвести, чтобы отмеченный фрагмент одного файла привести в соответствие с другим файлом, указываются командой diff3 одним из следующих способов: 

  f: n1 a 
     Добавить текст в файл f, где f = 1, 2 или 3, после
     строки n1.
  f: n1,n2 c 
     Изменить текст в строках с n1  по n2. Если  n1=n2,
     то запись может быть сокращена до n1.

Исходный вид изменяемых строк приводится сразу после символа c. Если два файла совпадают, то вид изменяемых строк приводится только для файла с большим номером.

ОПЦИИ
 
-e Сгенерировать набор команд для редактора ed, который добавляет к файлу1 отличия между файлом2 и файлом3 , то есть фрагменты, которые в нормальном режиме помечаются как ==== и ====3.
-x То же что и -e, но для фрагментов, помечаемых, как ====.
-3 То же что и -e, но для фрагментов, помечаемых, как ====3.

СМ.ТАКЖЕ
  cmp, comm, diff, ed.

КОДЫ ЗАВЕРШЕНИЯ
 
0    Различий нет.
1    Различия есть.
2    При выполнении команды зафиксированы ошибки.
