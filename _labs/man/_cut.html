cut
  удаление секции из каждой строки файла  

СИНТАКСИС
  
cut [опции...] [файл...]

ОПИСАНИЕ
 

Команда cut выводит заданную часть каждой строки файла в стандартный вывод. Команда используется для выборки колонок из таблицы или полей из каждой строки файла. Если не указано ни одного файла или задано имя -, используется стандартный ввод.


ОПЦИИ
 
-fсписок  выводить только эти поля.
Список представляет собой разделенный запятыми список диапазонов, перечисленных в порядке их возрастания. Диапазон может быть задан в одной из форм:
n - каждое n-е поле строки;
n- - от n-го поля до конца строки;
n-m - от n-го поля до m-го поля включительно;
-m - от начала строки до m-го поля включительно.
-dсимвол задание символа - разделителя полей.
