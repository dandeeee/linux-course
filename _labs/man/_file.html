file
  определение типа файла  

СИНТАКСИС
  
file файл...

ОПИСАНИЕ
  Команда file по первым 512 байтам пытается классифицировать файл, пытаясь определить тип файла, язык программирования, язык написания текстовых файлов. Информацию, используемую для классификации команда получает из файла: /usr/share/magic.

Вот примеры некоторых классификационных сообщений, выдаваемых командой file:

  • directory
  • character special
  • block special
  • symbolic link to ...
  • Bourne shell script text
  • C shell script text
  • Korn shell script text
  • awk commands text
  • perl commands text
  • compiled Java class data
  • C program text
  • C++ program text
  • Linux kernel
  • ELF 32-bit LSB executable
  • MS Windows PE
  • MS-DOS executable
  • English text
  • ASCII text
  • Microsoft Word Document
  • Microsoft Office Document
  • PDF document
  • PostScript document text
  • HTML document text
  • XML document text
  • LaTeX document text
