Вариант 1

Для файла (файлов) с заданным именем определить, является ли он символической ссылкой. Если да - проследить всю цепочку ссылок от этого имени.

Формат вызова: 

  команда файл...

Предполагается, что цепочка ссылок не выходит за пределы того каталога, в котором находится файл. Рекомандуется для проверки программы использовать файлы из каталога /home/metod.

Тип файла - символической ссылки может быть проверен выполнением команды ls с опцией -l. Символические ссылки имеют в выдаче этой команды тип l.

Результат должен иметь примерно такой вид: 

    $ команда
    file1 -> file2 -> file2 -> file4
