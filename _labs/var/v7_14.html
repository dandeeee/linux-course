Вариант 14

По информации файлов /home/metod/query* вывести для заданного кода покупателя коды всех товаров, которые он когда-либо покупал.

Формат вызова: 

  команда код_покупателя...

Нужно выполнить соединение (join) файлов - таблиц базы данных, для того, чтобы выбрать продажи, относящиеся к конкретным покупателям. А выбрать записи, относящиеся к заданному покупателю можно либо используя редактор sed, либо соединением с файла, состоящим из единственной строки - кода покупателя.
