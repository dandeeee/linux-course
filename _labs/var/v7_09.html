Вариант 9

По информации файлов /home/metod/query* определить среднюю зарплату для заданной должности.

Формат вызова: 

  команда должность...

Для выборки из /home/metod/query1 только строк, связанных с заданной должностью, можно, например, вывести название должности в отдельный файл, а затем выполнить его соединение (join) с файлом /home/metod/query1.
