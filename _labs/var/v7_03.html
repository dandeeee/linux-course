Вариант 3

Подсчитать количество строк, содержащих заданное слово в заданном файле. Если файл имеет тип, отличный от "ISO-8859 text", подсчет не производить, а просто вывести сообщение об этом.

Формат вызова: 

  команда файл...

Рекомендуется для проверки программы использовать каталог /home/metod.

Для проверки типа файла используйте команду file, для подсчета числа слов - wc.
