Вариант 2

Проверить заданный каталог на наличие в нем неразрешенных мягких ссылок. Если таковые имеются - отправить о них сообщение по mail пользователю derev. 

  команда каталог...

Рекомендуется для проверки программы использовать каталог /home/metod.

Тип файла - символической ссылки может быть проверен выполнением команды ls с опцией -l. Символические ссылки имеют в выдаче этой команды тип l.

Если файл является символической ссылкой, то неразрешенная ссылка может быть проверена попыткой чтения файла. Чтение из файла - неразрешенной ссылки приводит к сообщению "No such file". Но имейте в виду, что попыткой чтения может оказаться также неудачной из-за отсутствия у Вас прав доступа к этому файлу.

Отправка сообщения может быть обеспечена перенаправлением стандартного ввода команды mail на файл.
